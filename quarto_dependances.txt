# Quarto Dependencies – Python & Julia Climate Project

This document summarizes all required dependencies to run a **Quarto-based climate/meteorological data analysis project** using **Python and Julia** (ERA5 / NetCDF workflow).

---

## 1. Core Requirement: Quarto

Quarto is the main tool used to render reproducible reports (HTML, PDF, Word).

Install Quarto:
[https://quarto.org/download/](https://quarto.org/download/)

Check installation:

```
quarto --version
```

Note: Pandoc is already included with Quarto.

---

## 2. Python Environment

### Python Version

Recommended:

* Python ≥ 3.9

Check:

```
python --version
```

### Jupyter (Required for Quarto execution)

```
pip install jupyter
```

Check:

```
jupyter --version
```

---

## 3. Required Python Libraries (Climate & Geospatial)

Install everything at once:

```
pip install xarray netcdf4 numpy pandas matplotlib geopandas rioxarray shapely cartopy cdsapi
```

Library roles:

* xarray → read and process NetCDF (.nc) files
* netcdf4 → NetCDF backend
* numpy → numerical computations
* pandas → time series handling
* matplotlib → plotting
* geopandas → spatial masks (France shapefile)
* rioxarray → raster spatial clipping
* shapely → geometry operations
* cartopy → map visualization
* cdsapi → access Copernicus ERA5 API

---

## 4. Julia Environment

### Install Julia

[https://julialang.org/download/](https://julialang.org/download/)

Check:

```
julia --version
```

### Recommended Julia Packages

Inside Julia:

```
using Pkg
Pkg.add(["NetCDF", "NCDatasets", "DataFrames", "Plots", "Statistics", "GeoData", "ArchGDAL"])
```

Package roles:

* NetCDF / NCDatasets → read .nc climate files
* DataFrames → tabular data
* Plots → visualization
* Statistics → mean, anomaly, etc.
* GeoData / ArchGDAL → spatial processing

---

## 5. PDF Export (Optional but Recommended)

If PDF output is required:

### Windows:

* Install MiKTeX → [https://miktex.org](https://miktex.org)

### macOS:

* Install MacTeX → [https://tug.org/mactex/](https://tug.org/mactex/)

Check:

```
pdflatex --version
```

Without LaTeX, PDF rendering will fail.

---

## 6. Quarto Kernel Configuration

### Python in Quarto (.qmd):

```yaml
execute:
  kernel: python3
```

### Julia in Quarto:

```yaml
execute:
  kernel: julia
```

Make sure the Julia kernel is available in Jupyter:

```
julia
using IJulia
```

## 8. Typical Project Workflow

1. Download ERA5-Land data with cdsapi
2. Load NetCDF with xarray (Python) or NCDatasets (Julia)
3. Apply France spatial mask
4. Compute spatial averages
5. Compute climatology and anomalies
6. Plot results
7. Render final report with Quarto (HTML/PDF)
